## üöÄ Project Overview
This repository bridges the gap between **Abstract Mathematics** (Linear Algebra, Calculus) and **Applied Machine Learning**. 

Unlike standard projects that rely on `scikit-learn` abstractions, this codebase implements core algorithms **from first principles** using only Python and NumPy. The goal is to demystify the "black box" of AI by building the mathematical engine layer-by-layer.
